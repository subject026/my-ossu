Used to express requirements but also help clarify them.

Help discover connections between objects.

```

As a ___,  I want to  ___ so that ___.

```

`so that...` clause can be omitted if benefits are clear and generally known

**Nouns** usually correspond to objects in you software
`product`, `item`, `shopping cart`, `order` etc

**Verbs** can help you identify requirements that your objects may have as well as connections between objects
`add`, `remove`, `purchase`, `check`, `view` etc
